Detailed Description

template<class Type = ComplexDP>
class NoisyQureg< Type >

Class that expand QubitRegister states by adding noise between "logical" gates.

Parameters
num_qubitis the number of qubits When we refer to ``experimental'' gates, it means that noise gates are excluded. For the simulation to be faithful (i.e. with one-to-one correspondence with the experimental gates), we include among the experimental gates both the gates for the algorithm and those to schedule it according to the connectivity of the specific hardware.
