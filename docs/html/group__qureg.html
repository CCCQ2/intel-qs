template<class Type >
void QbitRegister< Type >::expectationValue ( std::vector< unsigned > &  qubits,
std::vector< unsigned > &  observables,
BaseType &  sum,
BaseType  coeff = 1. 
)

Compute expectation value of a Pauli string for multiple qubits over the full-register state.

Parameters
qubitsvector of the involved qubit indices
observablesvector of the involved pauli operators {1,2,3} -> {X,Y,Z}
sumcontains the initial value to which the expectation value will be added
coeffscalar coefficient that multiplies the expectation value

At the end of the function, the variable 'sum' is updated via:
sum –> sum + coeff * <psi| Pauli_String |psi>
where the Pauli_String is defined by:
observables[0]_qubit[0] . observables[1]_qubit[1] ...
and each Pauli operator is encoded according to:
{1,2,3} –> {X,Y,Z}
